Rapportages
======================

Reports are saved search queries. A user can create a ``Rapportage`` through `Zoeken op zaak <https://zac.cg-intern.utrecht.nl/ui/zaak-starten>`_ and ticking the "Zoekopdracht opslaan als rapportage" box.
